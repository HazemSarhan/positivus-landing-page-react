## Installation :

1. Clone the repository:

```sh
git clone https://github.com/HazemSarhan/positivus-landing-page-react.git
```

2. Navigate into the project directory:

```sh
cd positivus-landing-page-react
```

3. Install dependencies:

```sh
npm install
```

6. Start the server:

```sh
npm run dev
```
